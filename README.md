# Atividades_Versionamento
# 🔗 Encurtador de URLs em Python

Este é um projeto simples e funcional desenvolvido em Python para transformar URLs longas em links curtos e fáceis de compartilhar, utilizando a biblioteca `pyshorteners`.

## 🎯 Objetivo
O objetivo deste projeto é facilitar o compartilhamento de links extensos, integrando o script com serviços de encurtamento (como o TinyURL) via API, demonstrando o uso de bibliotecas externas e manipulação de entradas do usuário.
